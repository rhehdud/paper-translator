# 논문을 읽어요

매주 arXiv에서 5개 분야(`cs.CL`, `cs.CV`, `cs.LG`, `cs.AI`, `cs.RO`)에서 화제가 된 논문을 1편씩 골라 한국어로 전문 번역해 쌓아두는 아카이브입니다.

- 수집: arXiv API
- 선별: Hugging Face Daily Papers 업보트 (매칭 없으면 최신순)
- 추출: [Marker](https://github.com/datalab-to/marker)
- 번역: NVIDIA NIM `google/gemma-4-31b-it`
- 발행: 이 사이트 (GitHub Actions로 매주 자동 생성)

왼쪽 또는 위 네비게이션에서 분야별 번역 목록을 확인하세요.
